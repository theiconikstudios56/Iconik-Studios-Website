11. SEO Preparation Notes
Page: All Pages  |  General SEO guidance for when keywords are provided.
The following framework outlines where to place keywords once they are provided by the client.
On-Page SEO — Placement Priority
H1 Tags: Each page should have exactly one H1 containing the primary keyword for that page.
H2 Tags: Sub-sections should use related keywords and question-based phrases.
Meta Titles: ~60 characters. Format: Primary Keyword | Brand Name
Meta Descriptions: ~155 characters. Include the primary keyword naturally. CTA-driven.
First 100 Words: The primary keyword should appear in the first paragraph of every service page.
Alt Text: All images should have descriptive alt tags including relevant keywords.
Internal Links: Each page should link to at least 2 other pages on the site using descriptive anchor text.
